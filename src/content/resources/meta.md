I got sick of having to Google around to find all the meta tags I can't remember. So I made a list of the ones I
frequently use. In my travels, I did find [this website](https://ogp.me) that helped me out a lot.
AAnything OG tags I don't have below should be on that website.

**Behold!**

```html
<!-- General Meta Tags -->
<meta name="description" content="" />
<meta name="author" content="" />

<meta property="og:title" content="" />
<meta property="og:description" content="" />
<meta property="og:type" content="website" />
<meta property="og:image" content="" />
<meta property="og:url" content="" />

<!-- Twitter Meta Tags -->
<meta name="twitter:card" content="summary_large_image" />
<meta name="twitter:title" content="" />
<meta name="twitter:description" content="" />
<meta name="twitter:image" content="" />

<!-- Get rid of those pesky ai hopefully... -->
<meta name="robots" content="noai, noimageai" />
```
